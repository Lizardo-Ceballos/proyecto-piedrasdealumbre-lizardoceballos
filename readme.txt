//Resumen
Este proyecto es una web para la empresa "Piedras de Alumbre MID".
Su principal objetivo es ofrecer a sus clientes una carta presentación que evoque confianza y seriedad para la marca
Su principal función es actuar como un medio de información para el cliente acerca de la empresa, sus productos, términos y condiciones, etc.
Esta pagina también debe funcionar como un medio para captar leads y poner en contacto a la empresa con sus clientes
Por ultimo la pagina debe funcionar como un medio de redirección a las diferentes redes sociales y puntos de venta electrónicos de la empresa

//Construcción
Para le implementación de esta pagina estática se opto por utilizar las tecnologías estándar en la industria como lo son: HTML, CSS y SASS todo apoyado por un control de versiones en Git y resguardado en un repositorio de GitHub
De igual forma se decidió utilizar como framework Bootstrap en su versión 5.1.3 para implementar diferentes funcionalidades sin el uso de JS en todas las paginas.
Por ultimo en las animaciones de los elementos se decidió utilizar la librería AOS con la intención de agregar dinamismo a las paginas.
